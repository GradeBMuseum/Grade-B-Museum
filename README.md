# Grade-B Museum

쓸데없는 기능들을 진지하게 구현한 개인 실험 프로젝트입니다.

Grade-B Museum은 실용성을 의도적으로 배제한 기능들을  
‘전시물’이라는 형태로 구현하는 개인 프로젝트입니다.

이 프로젝트의 목적은 문제 해결이 아니라,  
불필요한 기능을 얼마나 정교하게 설계하고 구현할 수 있는지를  
보여주는 데 있습니다.

쓸모는 없지만,  
인터랙션 설계·상태 관리·애니메이션 구현만큼은 진지하게 접근합니다.

---

## Philosophy

- 이 프로젝트는 **문제를 해결하지 않습니다**
- 실무에서는 만들지 않을 기능을 일부러 설계합니다
- 하나의 전시물은 하나의 아이디어만 담습니다
- 구현의 완성도는 타협하지 않습니다

> Useless by design.  
> Serious by implementation.

---

## Project Status

- 현재 전시는 준비 중입니다
- 이 프로젝트는 **완성형이 아닌 지속적으로 확장되는 전시관**입니다
- 심심할 때마다 새로운 전시가 추가됩니다

---

## Tech Stack

- Flutter (Web)
- Dart

---

## Open Source Policy

이 프로젝트는 오픈소스로 공개되어 있습니다.  
다만, 전시물의 구현은 프로젝트 큐레이터(작성자)가 직접 진행합니다.

- 코드 기여(PR)는 받지 않습니다
- 전시 아이디어 제안은 Issue로 환영합니다
